GETTING STARTED

Account and access

Can I use the same account on the app and website?

Yes. Sign in with the same Firebase email/password or Google account. Your Firebase user ID connects the website to the wardrobe and saved outfits created in the mobile app.

Why can’t I sign in with Google?

Allow pop-ups, confirm you are using the same Google account as the app, and try a fresh browser tab. The website domain must also be authorized in Firebase Authentication. If the problem continues, send the displayed Firebase error code to support.

Does closing the browser sign me out?

No. ClothMatics uses browser-local Firebase session persistence. Use the Sign out button on a shared device.

WARDROBE

Your clothes and looks

Can I add clothes through the website?

No. Website wardrobe access is intentionally read-only. Add, extract, edit, or delete garments through the ClothMatics mobile app.

Why is my website wardrobe empty?

Make sure you signed in with the same account used in the mobile app. If the account is correct, refresh after confirming that the app finished saving the items.

Are favorites and saved outfits synchronized?

Yes. The website reads favorites, Lookbook items, and saved outfits from the same Firestore project used by the mobile app.

Can the website change laundry status?

No. The website can filter and use laundry status while styling, but changes must be made in the mobile app.

AI STYLIST

Outfit generation

How does the web outfit generator work?

Choose an occasion and optional preference. ClothMatics sends relevant wardrobe metadata to a protected Cloudflare server function, which requests one outfit using only item IDs from your wardrobe.

Does web outfit generation send my garment photos to AI?

No. The website generator sends descriptive metadata such as category, color, material, fit, season, occasion, and laundry status. It does not send garment image bytes.

Why did the AI choose an unavailable item?

Confirm the laundry status in the mobile app and try again. AI suggestions can be imperfect, so review every recommendation before wearing it.

Is there a daily generation limit?

A daily web limit may be applied to control cost and prevent abuse. The app may provide different limits depending on your plan.

PRIVACY & SUPPORT

Data and help

Who can see my wardrobe?

Firestore Security Rules restrict normal queries to the authenticated owner. A specifically authorized administrator may access operational data for support, security, and service administration.

How do I request my data or delete my account?

Email chiragsharma376@gmail.com from your registered account and use “Privacy Request” in the subject. See the Privacy Policy for details.

Where can I report a problem?

Email chiragsharma376@gmail.com with your device, browser or app version, the action you attempted, and any visible error code. Never send your password or API keys.
